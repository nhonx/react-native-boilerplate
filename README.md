## react-native-boilerplate
 **A useful template for quickly creating a React-Native app.**
  - Clear code-organizing: App - Components - Styles - Constants.
  - Splashscreen supporting.
  - Side-menu enabled.
  
 **Package using:**
  - React
  - React Navigation.
